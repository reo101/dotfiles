# reo101's Dotfiles

Managed with [`chezmoi`](https://github.com/twpayne/chezmoi)

# Dependencies
- git
- curl
- zsh
- [starship](https://starship.rs/)

# Usage

## Installation

```zsh
sh -c "$(curl "https://raw.githubusercontent.com/reo101/dotfiles/master/install.sh")"
```

## Update

```zsh
chezmoi update
```
